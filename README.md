# SO-Snake-Windows

### Jogo Snake para a disciplina de Sistemas Operacionais no Windows

## Instruções:

- Baixe ``` setup.exe ``` e ```Instalador-Snake-SO.msi```
- Clique duas vezes no arquivo ``` setup.exe ``` 
- Para jogar inicie o programa ```Snake-SO``` no seu Desktop
